# 📈 智能盯盘系统

一个部署在 GitHub Pages 上的个人股票持仓监控面板，支持实时行情、K线分析、技术指标、资金流向、市场情绪等功能。

## ✨ 功能特性

| 模块 | 说明 |
|------|------|
| 📋 持仓总览 | 显示全部持仓股票的当前价格、涨跌幅、买卖信号 |
| 📊 K线走势 | 日/周/月线K线图，含MA5/MA10/MA20均线 |
| 🔬 技术指标 | RSI、MACD、布林带、PE/PB、换手率、量比 |
| 💰 资金流向 | 特大/大/中/小单净流入，识别主力动向 |
| 🤖 AI购买建议 | 综合技术指标计算买入/持有/卖出信号 |
| 🌡️ 市场情绪 | 持仓多空比、大盘指数走势 |
| 🏛️ 大盘指数 | 上证/深证/创业板/沪深300/中证500/上证50 |
| 🧲 北向资金 | 沪股通+深股通每日净流入数据 |
| 📰 关键资讯 | 财联社实时财经快讯 |
| 🌙 主题切换 | 暗色/亮色主题一键切换 |

## 🚀 快速部署到 GitHub Pages

### 方法1：直接上传（推荐）

1. 在 GitHub 创建新仓库（或使用现有仓库）
2. 上传 `index.html` 到仓库根目录
3. 进入仓库 **Settings → Pages**
4. Source 选择 `main` 分支 `/ (root)`
5. 保存后等待 1-2 分钟，访问 `https://<你的用户名>.github.io/<仓库名>/`

### 方法2：Git 命令行

```bash
git init
git add index.html README.md
git commit -m "feat: 初始化智能盯盘系统"
git remote add origin https://github.com/<你的用户名>/<仓库名>.git
git push -u origin main
```

然后在 GitHub 仓库设置中开启 Pages。

## ⚙️ 自定义持仓

打开页面后，点击右上角 **＋** 按钮添加股票，输入：
- 股票代码（如 `000547.SZ`）
- 股票名称（如 `大北农`）
- 买入成本（可选）
- 持有数量（可选）

数据存储在浏览器 `localStorage` 中，刷新不丢失。

## 📡 数据来源

行情数据通过 [CodeBuddy 金融数据接口](https://www.codebuddy.cn/) 获取，包含：
- A股历史日/周/月线行情
- 每日基本面指标（PE/PB/换手率等）
- 个股资金流向
- 大盘指数行情
- 北向资金流向
- 财经新闻快讯
